## HC掌上物业是完全开源并免费商用的手机版小区物业管理系统，
## 包含工单维修,巡检打卡,采购,物品领用,投诉处理,房产管理,通讯录,公告和小区广告等功能,
## 项目采用uni-app开发一次编码多端运行欢迎加群交流857791253
